## Exact Commands to Run

### Step A вЂ” Create virtual environment
```bash
cd university-ai-platform/ai-service

# Create venv (recommended to isolate deps)
python -m venv venv

# Activate:
source venv/bin/activate        # Linux/Mac
venv\Scripts\activate           # Windows
```

### Step B вЂ” Install dependencies
```bash
pip install -r requirements.txt
```

This downloads (~800MB first run):
- `sentence-transformers` вЂ” downloads `all-MiniLM-L6-v2` embedding model (~90MB)
- `faiss-cpu` вЂ” vector search library
- `PyMuPDF` вЂ” PDF text extraction
- `langdetect` вЂ” language detection
- `google-generativeai` OR `openai` вЂ” LLM client

### Step C вЂ” Configure environment
```bash
cp .env.example .env
```

Edit `.env`:
```env
LLM_PROVIDER=gemini            # or "openai"
GEMINI_API_KEY=your_key_here   # get from console.cloud.google.com
# OR:
OPENAI_API_KEY=your_key_here
```

Get a free Gemini API key: https://aistudio.google.com/app/apikey

### Step D вЂ” Start the service
```bash
python main.py
# OR for auto-reload during development:
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

Service starts at: **http://localhost:8000**
Interactive API docs: **http://localhost:8000/docs**

### Step E вЂ” Run unit tests (no API key needed)
```bash
pytest tests/test_pipeline.py -v
```

## Testing the AI Service Directly

```bash
# Health check
curl http://localhost:8000/health

# Upload a PDF
curl -X POST http://localhost:8000/upload-pdf \
  -F "file=@/path/to/university_handbook.pdf"

# Trigger reindex
curl -X POST http://localhost:8000/reindex

# Ask a question
curl -X POST http://localhost:8000/ask \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the fee structure?", "language": "auto"}'
```

Expected `/ask` response:
```json
{
  "answer": "The annual tuition fee for B.Tech CSE is INR 85,000...",
  "confidence": 0.847,
  "detected_language": "en",
  "language_name": "English",
  "source_chunks": 3,
  "fallback": false
}
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| `ModuleNotFoundError: fitz` | `pip install PyMuPDF` |
| `ModuleNotFoundError: faiss` | `pip install faiss-cpu` |
| `GEMINI_API_KEY not set` | Add key to `.env` file |
| Model download hangs | Check internet connection; `all-MiniLM-L6-v2` is ~90MB |
| `/ask` returns fallback always | No PDFs uploaded yet вЂ” upload and call `/reindex` first |
| Port 8000 in use | `lsof -ti:8000 \| xargs kill` |
